# MaidFinder

This repository now contains the **Next.js** version of MaidFinder.

## Project location

- Main app folder: `nextjs/`

## Run locally

```bash
cd nextjs
npm install
npm run dev
```

Open: `http://localhost:3000`

## Build

```bash
cd nextjs
npm run build
npm start
```

## Deploy to Vercel

When importing on Vercel, set:

- **Root Directory**: `nextjs`
- **Build Command**: `npm run build`
